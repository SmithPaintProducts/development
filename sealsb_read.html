Base Boost

Smith’s Base Boost is a blend of silicates used as an additive to Smith’s Color Floor and/or Smith’s Color Wall to promote substrate adhesion for base or primary colors as well as increase the stains durability. Smith’s Color Floor and Color Wall stains already achieve superior mechanical adhesion as a result of the modified acrylic carrier, but adding Base Boost into the Smith’s Concentrate produces an additional bond by reacting with the unhydrated calcium hydroxide found in the concrete. Smith’s Base Boost should not be used for application to overlays, previously coated substrates, or as an additive for highlight colors.
